# Roman Society 1.1.325

Release date: 2026-06-16

A gameplay-logic correctness pass: the character interaction menu let the player
target themselves and their own close family with social actions, and let them court
blood relatives. These are fixed both in the menu (the options no longer appear) and in
the action handlers (defensive guards, so a stale menu or console call cannot bypass them).

## Self-Targeting Fixed

When viewing your own current character, the menu no longer offers actions that only
make sense against other people:

- "Praise in public" is hidden on yourself.
- "Spread rumor" is hidden on yourself (it was already hidden for spouse/close family).
- "Court privately" / romance is hidden on yourself.
- The social-visit action ("invite home", "household council", etc.) is hidden on yourself.

`praisePerson` and `spreadRumor` now also no-op if the target resolves to the current
character.

## Incest / Close-Family Courtship Fixed

- "Court privately" no longer appears for close family — children, parents, siblings,
  grandparents/grandchildren, or in-laws. Only the spouse keeps a dedicated marital
  route ("Lie together").
- `courtCharacter` and `resolveCourtship` (where rapport, romance, and pregnancy are
  actually applied) now reject self and close-family targets, so the romance/pregnancy
  path can never fire on the player's own children or other close kin even via a stale
  menu button.

## Other Logic Guards

- `adoptHeir` now refuses to adopt/designate yourself or your own spouse as heir.
- Verified existing systems were already safe: crime denunciation excludes the player's
  own house; the Coemptio matchmaker and marriage acceptance already block the same
  dynasty (including cadet houses) and close blood kin via `isMarriageCompatible`; the
  Attempt Murder bundle already excludes the current character.

## Packaging

- Bumped all module mixin version markers and the engine version to `1.1.325`.
- Updated root and runtime README files to `1.1.325`.
- Release artifact: `Releases/CORSociety-v1.1.325.zip`.

## Remaining Validation

- Local object-literal syntax checks validate every module file, but the interaction
  menu should still be spot-checked inside Citizen of Rome (open your own character, a
  child, a spouse, and an outsider) because the local workspace does not run the full
  game runtime or DAAPI UI.
