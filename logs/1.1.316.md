# Roman Society 1.1.316

## 1.1.316 - Bundle the CoR Translator inside Roman Society

The `cor_translator` mod is now shipped as a bundled sub-mod of Roman Society, the
same way Bank of Rome, Coemptio, Household Slaves, Play As, etc. are bundled.

### Added

- `cor_society/bundled/cor_translator/` (main.js, monthly.js, README.md, style.css,
  icon.svg). Installing Roman Society now also installs the translator; no separate
  download is needed.

### How it was integrated

- All event paths were rewritten from `/cor_translator/main` to
  `/cor_society/bundled/cor_translator/main`, and all images from
  `/cor_translator/icon.svg` to `/cor_society/bundled/cor_translator/icon.svg`, so
  the game loads it from inside the Society package.
- `monthly.js` points at `['/cor_society/bundled/cor_translator/main']`, matching the
  other bundled mods, so the game auto-discovers and boots it.
- The translator keeps its own global action ("CoR Translator"), its own internal
  version (`1.0.6`), and its existing localStorage / global-flag cache keys
  (`corTranslator.v1.*`, `corTranslatorConfig`, ...), so any cache a player already
  built carries over unchanged.

### Includes the translator fixes from 1.0.6

- De-duplicated network path (`fetchTranslationOnce`): the modal pretranslation and
  the scan queue can no longer request the same text at the same time.
- `failedAt` backoff respected by every path; lighter scanning cadence.

### Notes

- The standalone `CORmods/cor_translator` copy is left untouched. If you load BOTH
  the standalone and the bundled copy, they would register the same global-action
  key twice; install only one (the bundled one ships with Society).
- Runtime version aligned to `1.1.316`. The bundled translator's own version string
  (1.0.6) is independent of the Society version and is intentionally not rewritten.
