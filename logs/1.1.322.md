# Roman Society 1.1.322

Release date: 2026-06-16

## Full Plan Integration

- Added the advanced Roman systems layer from `PLANDEMEJORA.md`: crimes, courts, prisons, tax fraud, accusations, escapes, pardons, cursus honorum, clientela, legions, coalitions, dictatorship pressure, imperial laws, spies, hooks, adoption, cognomina, religion pressure, property crises, and bounded monthly simulation.
- Added a module manifest and technical safety layer so advanced systems migrate, prune, and validate their Society-owned data without changing unrelated vanilla save structures.
- Added law descriptions and tooltips for all Society Roman power laws.

## Menu And UX Changes

- Moved `Crimes, courts, and prison` into the Society hub immediately below Bank of Rome, keeping it available outside the Senate menu.
- Removed the old Monthly life summary box from the Society hub.
- Updated private loan amount actions so the visible amount comes from the game stat-change icon, while acceptance chance and repayment details stay in the tooltip.

## Vanilla Compatibility

- Legionary sentences now reuse vanilla-style away/busy/military-service flags and the vanilla military/war icon path where available.
- The mod does not create a duplicate global war just to support a sentence; it only marks the sentenced character with Society-owned military sentence metadata and vanilla-compatible character status.

## Slavery Fixes

- Fixed slave-market purchases so bought slaves are registered in `playerSlaves`, linked to the player house, and kept visible for household-slave actions.
- Preserved pending real slave records when the character object is temporarily unavailable instead of dropping the record.
- Added a slave-specific character menu so enslaved characters do not expose the same free-citizen action flow as normal characters.

## Wardrobe Fixes

- Fixed wardrobe portrait mutation by cloning portrait/look data before applying outfits.
- Tightened roster portrait replacement so changing the player outfit does not overwrite a spouse or another character with a matching nearby portrait block.

## Packaging

- Updated root and runtime README files to `1.1.322`.
- Release artifact: `Releases/CORSociety-v1.1.322.zip`.

## Remaining Validation

- Local object-literal syntax checks can validate the module files, but final balance and UI flow should still be playtested inside Citizen of Rome because the local workspace does not run the full game runtime or DAAPI UI.
